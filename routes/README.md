# CRUD Application

## Overview

This is a simple CRUD (Create, Read, Update, Delete) application that allows you to manage entities such as Students and Courses. The application is built using either **Node.js with Express** or **FastAPI with Python**. Choose the implementation that best fits your needs.

## Technologies Used

- **Node.js (Express)** or **FastAPI (Python)**
- **MongoDB** for database management
- **Mongoose** (for Node.js) or **PyMongo** (for FastAPI)

## Setup Instructions

### Node.js (Express) Implementation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>  # Replace <repository-url> with your GitHub repository URL
   cd crud-app  # Change directory to the project folder
   ```
2. Install dependencies:
   npm install

Run the server:
node server.js

Open your browser:
http://localhost:3000
